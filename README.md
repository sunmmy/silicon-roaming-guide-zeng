# 个人网站项目

这是一个用于建设个人展示网站的工作目录。网站目标是集中呈现个人介绍、项目作品、研究/写作内容、联系方式，以及后续可持续更新的个人档案。

## 当前结构

- `AGENT.md`：核心协作规则，后续 Agent 进入项目时优先阅读。
- `docs/`：项目背景、进展、下一步、技术方案等简洁工作文档。
- `content/`：未来网站会展示的内容素材，例如个人介绍和项目条目。
- `assets/`：图片、截图、头像、项目封面等静态素材。
- `src/`：网站页面、布局和样式代码。
- `public/`：会被网站直接访问的公开静态资源。

## 本地运行

已经搭建了 Astro 第一版网站。

```bash
npm run dev
```

本地访问地址：

```text
http://127.0.0.1:4321/
```

正式发布前检查：

```bash
npm run build
```

## 推荐工作方式

1. 先在 `content/` 补齐个人介绍和项目资料。
2. 再根据 `docs/tech-plan.md` 搭建网站代码。
3. 每完成一次重要修改，更新 `docs/progress.md` 和 `docs/next-steps.md`。
