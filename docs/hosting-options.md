# 域名与托管方案

## 结论

个人展示网站第一版不建议购买服务器。推荐使用静态网站：代码生成 HTML/CSS/JS 文件，再托管到稳定平台。

当前第一版已经部署到 Cloudflare：

- https://silicon-roaming-guide-zeng.zengli100.workers.dev

这个地址可作为临时线上访问地址，后续再绑定正式域名。

## 推荐路线

### 路线 A：稳定优先，适合国内外访问

- 域名：在阿里云或腾讯云购买，方便后续备案。
- 托管：阿里云 OSS 静态网站托管或腾讯云静态网站托管。
- 加速：开启 CDN，选择全球或中国内地 + 海外加速。
- 合规：如果网站托管在中国内地，需要先完成 ICP 备案；ICP备案完成后，通常还要按要求完成公安联网备案。

优点：

- 国内访问更稳定。
- 海外也可以通过云厂商 CDN 改善速度。
- 不用维护服务器。

代价：

- 需要实名和备案。
- 上线流程比纯海外平台多几步。

### 路线 B：最快上线，适合先做出第一版

- 域名：Cloudflare、Namecheap、阿里云或腾讯云均可。
- 托管：Cloudflare Pages、Vercel 或 Netlify。
- 合规：不使用中国内地服务器时，一般不需要 ICP 备案。

优点：

- 上线快。
- 免费额度通常足够个人网站早期使用。
- 适合快速验证设计和内容。

代价：

- 国内访问速度和稳定性不保证。
- 如果以后非常重视国内访问，可能仍要迁移到国内托管并备案。

## 我的建议

先按路线 B 把网站做出来，尽快看到真实网页；同时准备域名和备案资料。等内容稳定后，再切换到路线 A 作为正式长期方案。

这样做的好处是：前期不用卡在备案流程里，后期也不会被临时方案锁死。

## 购买清单

第一阶段：

- 域名：建议购买 `.com`，如果合适也可以考虑 `.me` 或个人名字相关域名。
- 托管：先用 Cloudflare Pages / Vercel / Netlify。

长期正式阶段：

- 域名：保留同一个域名。
- 静态托管：阿里云 OSS 或腾讯云静态网站托管。
- CDN：按流量计费即可，个人网站通常流量不大。
- 备案：ICP备案 + 公安联网备案。

## 小白注意事项

- 域名要看续费价格，不只看首年优惠。
- 不要买太多后缀，先买一个主域名即可。
- 不要先买服务器，个人展示站第一版用不上。
- 域名、托管、CDN 尽量用同一家云厂商，后续配置更简单。
- 网站内容确认前，不要急着做复杂功能。
